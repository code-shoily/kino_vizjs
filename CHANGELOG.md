## 0.5.0

### Added
- Initial Hex release.
- `Kino.VizJS` component for rendering GraphViz DOT strings via Viz.js.
- `Kino.VizJS.SmartCell` interactive smart cell for Livebook.
- Theme-aware rendering (adapts to Livebook light/dark mode).
